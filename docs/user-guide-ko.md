# StockFlow 사용자 안내서 초안

## 1. StockFlow란?

StockFlow는 매장 재고와 발주 업무를 관리하기 위한 웹앱입니다.

주요 목적은 다음과 같습니다.

- 남은 재고를 빠르게 확인합니다.
- 바코드 스캔으로 입고, 출고, 이동, 조정 작업을 기록합니다.
- 부족 재고와 프레시 품목을 따로 확인합니다.
- 작업 로그로 누가 어떤 재고 작업을 했는지 확인합니다.
- 매장별 직원과 관리자 권한을 관리합니다.

## 2. 계정과 권한

StockFlow에는 세 가지 권한이 있습니다.

### 마스터 계정

마스터 계정은 전체 매장을 관리합니다.

할 수 있는 일:

- 새 매장 생성
- 전체 매장 확인
- 전체 사용자 확인
- 사용자를 매장에 배정
- 사용자 이름 수정
- 사용자 삭제
- 직원 초대 링크 생성

### 매장 관리자 계정

매장 관리자 계정은 본인 매장의 운영 설정과 직원을 관리합니다.

할 수 있는 일:

- 직원 관리
- 초대 링크 생성
- 상품 관리
- 카테고리 관리
- 품목 단위 관리
- 발주처 관리
- 환경설정 관리

### 직원 계정

직원 계정은 재고 작업과 조회 기능을 사용합니다.

할 수 있는 일:

- 홈 화면 확인
- 바코드 스캔
- 재고 작업
- 재고 현황 확인
- 부족 재고 확인
- 작업 로그 확인

## 3. 로그인과 회원가입

### 로그인

1. 앱 첫 화면에서 `로그인`을 누릅니다.
2. 이메일과 비밀번호를 입력합니다.
3. `로그인` 버튼을 누릅니다.

### 초대 링크로 회원가입

1. 관리자 또는 마스터가 보낸 초대 링크를 엽니다.
2. `초대 수락` 화면에서 초대받은 이메일을 확인합니다.
3. `회원가입` 버튼을 누릅니다.
4. 회원가입 화면에서 초대받은 이메일이 자동으로 입력되어 있는지 확인합니다.
5. 비밀번호와 비밀번호 확인을 입력합니다.
6. `회원가입` 버튼을 누릅니다.
7. 가입이 완료되면 자동으로 매장에 연결됩니다.

주의:

- 초대받은 이메일과 회원가입 이메일이 달라지면 매장 연결이 되지 않습니다.
- 초대 링크가 만료되었거나 이미 사용된 경우 관리자에게 새 초대 링크를 요청해야 합니다.

## 4. 하단 메뉴

앱 하단에는 주요 메뉴가 있습니다.

### 홈

오늘 입고된 품목, 내일 입고 예정 품목, To do list, 인수인계를 확인합니다.

### 스캔

바코드를 스캔하거나 상품명으로 검색합니다.

### 재고현황

전체 상품의 창고 수량, 매장 수량, 총재고, 상태를 확인합니다.

### 부족재고

최소재고 이하인 품목과 프레시 발주 품목을 확인합니다.

### 작업로그

입고, 출고, 이동, 조정 작업 기록을 확인합니다.

## 5. 바코드 스캔

1. 하단 메뉴에서 `스캔`을 누릅니다.
2. 카메라로 바코드를 스캔합니다.
3. 상품이 등록되어 있으면 해당 상품의 `재고 작업` 화면으로 이동합니다.
4. 상품이 등록되어 있지 않으면 `상품 등록` 화면으로 이동합니다.

검색으로도 상품을 찾을 수 있습니다.

## 6. 상품 등록

미등록 상품을 처음 스캔하면 상품 등록 화면이 열립니다.

입력 항목:

- 상품명
- 바코드
- 카테고리
- 보관 구분
- 발주처
- 품목 단위
- 최소 재고
- 링크

보관 구분은 여러 개 선택할 수 있습니다.

예:

- 냉장
- 냉동
- 상온
- 냉장 + 냉동

품목 단위 기본값:

- 박스
- 낱개
- 줄
- 팩

상품을 저장하면 바로 재고 작업 화면으로 이동합니다.

## 7. 재고 작업

재고 작업 화면에서는 한 상품의 재고를 변경합니다.

작업 종류:

- 조정
- 입고
- 이동
- 출고

### 조정

실제 재고 수량을 기준으로 재고를 맞춥니다.

예:

- 창고 재고가 실제로 10개라면 `창고` 선택 후 수량 `10` 입력
- 저장하면 창고 재고가 10개로 맞춰집니다.

### 입고

재고가 새로 들어왔을 때 사용합니다.

예:

- 매장에 5개 입고
- `입고` 선택
- `매장` 선택
- 수량 `5` 입력
- 저장

프레시 품목이 부족 재고에 올라와 있는 상태에서 입고 처리하면 부족 재고 화면에서 자동으로 제거됩니다.

### 이동

창고와 매장 사이에서 재고를 이동할 때 사용합니다.

예:

- 창고에서 매장으로 3개 이동
- `이동` 선택
- `창고 -> 매장` 선택
- 수량 `3` 입력
- 저장

### 출고

재고가 소진되었거나 빠져나갔을 때 사용합니다.

예:

- 매장에서 2개 출고
- `출고` 선택
- `매장` 선택
- 수량 `2` 입력
- 저장

## 8. 재고 작업 화면의 버튼

상단 아이콘 버튼:

- 뒤로가기
- 상품 수정
- 목록으로 이동
- 재고 작업 히스토리

수량 입력:

- `-` 버튼으로 수량 감소
- `+` 버튼으로 수량 증가
- 빠른 수량 버튼으로 수량 추가
- 직접 숫자 입력 가능

저장:

- 수량과 작업 내용을 확인한 뒤 `저장`을 누릅니다.

## 9. 재고 작업 히스토리와 되돌리기

재고 작업 화면의 히스토리 버튼을 누르면 최근 작업 기록을 확인할 수 있습니다.

히스토리에서 특정 시점으로 되돌릴 수 있습니다.

주의:

- 되돌리기는 재고 수량에 직접 영향을 줍니다.
- 되돌리기 전 상품명과 작업 시간을 확인해야 합니다.

## 10. 재고 현황

재고 현황 화면에서는 전체 상품을 확인합니다.

표시 정보:

- 상품명
- 보관 위치
- 발주처
- 품목 단위
- 창고 재고
- 매장 재고
- 총재고
- 최소재고
- 부족 여부
- 상품 링크

사용 방법:

1. 하단 메뉴에서 `재고현황`을 누릅니다.
2. 카테고리 버튼으로 품목을 필터링합니다.
3. 검색창에서 상품명 또는 바코드를 검색합니다.
4. 상품을 누르면 재고 작업 화면으로 이동합니다.

보기 방식:

- 간소화
- 전체

## 11. 부족 재고

부족 재고 화면은 최소재고 이하인 품목을 보여줍니다.

표시 대상:

- 총재고가 최소재고 이하인 품목
- 프레시 발주 품목으로 선택된 품목

### 발주 완료 체크

발주한 품목은 `발주 완료` 체크박스를 선택합니다.

### 프레시상품

`프레시상품` 버튼을 누르면 쿠팡 프레시 발주처 품목을 선택할 수 있습니다.

선택된 프레시 품목은 부족 재고 화면에 표시됩니다.

### 입고 완료

프레시 품목에는 `입고 완료` 버튼이 표시됩니다.

입고 완료를 누르면 해당 품목은 부족 재고 화면에서 제거됩니다.

### 입고완료 되돌리기

`입고완료 되돌리기` 버튼을 누르면 방금 입고 완료 처리한 품목을 다시 되돌릴 수 있습니다.

여러 번 누르면 최근 처리 순서대로 되돌립니다.

예:

- 한 번 누름: 직전 입고 완료 취소
- 두 번 누름: 두 단계 전까지 취소
- 세 번 누름: 세 단계 전까지 취소

### 긴급발주요청

긴급으로 발주해야 하는 품목은 `긴급발주요청`을 사용합니다.

사용 방법:

1. `긴급발주요청` 버튼을 누릅니다.
2. 품목을 선택합니다.
3. 발주 요청 수량을 입력합니다.
4. 요청 저장을 누릅니다.

## 12. 작업 로그

작업 로그 화면에서는 재고 작업 내역을 확인합니다.

확인 가능한 내용:

- 작업 시간
- 작업자
- 상품명
- 작업 종류
- 수량 변경 내용

작업 로그는 재고 변경 이력을 확인할 때 사용합니다.

## 13. 홈 화면

홈 화면에는 매장 운영에 필요한 요약 정보가 표시됩니다.

### 금일 입고품목

오늘 입고 처리한 품목을 보여줍니다.

금일 입고 기록을 삭제할 수 있습니다.

삭제한 금일 입고 기록은 최근 삭제 건에 한해 되돌릴 수 있습니다.

### 내일 입고예정 품목

내일 입고 예정으로 표시된 품목을 확인합니다.

### To do list

당일 할 일을 등록하고 완료 처리합니다.

### 인수인계

직원 간 전달할 내용을 기록합니다.

## 14. 최소재고 품목

상단 메뉴에서 `최소재고 품목`을 열 수 있습니다.

이 화면은 재고 작업에서 상태 기능을 켠 품목을 모아 보여줍니다.

상태:

- 충분
- 절반 이하
- 발주 필요

## 15. 관리자 메뉴

관리자 또는 마스터 계정은 왼쪽 상단 메뉴를 사용할 수 있습니다.

관리자 메뉴 항목:

- 상품 관리
- 카테고리 관리
- 품목 단위 관리
- 발주처 관리
- 직원 관리
- 환경설정

마스터 메뉴 항목:

- 전체 매장
- 전체 사용자

## 16. 상품 관리

상품 관리 화면에서는 등록된 상품을 관리합니다.

할 수 있는 일:

- 상품명 수정
- 카테고리 변경
- 링크 수정
- 상품 병합
- 상품 비활성화
- 비활성 상품 삭제

상품 병합은 같은 품목이 중복 등록되었을 때 사용합니다.

## 17. 카테고리 관리

카테고리 관리 화면에서는 상품 등록 시 사용하는 카테고리를 관리합니다.

할 수 있는 일:

- 카테고리 추가
- 이름 수정
- 순서 변경
- 비활성화
- 삭제

활성 카테고리는 삭제할 수 없습니다.
삭제하려면 먼저 비활성화해야 합니다.

## 18. 품목 단위 관리

품목 단위 관리 화면에서는 상품 등록 시 사용하는 단위를 관리합니다.

기본 단위:

- 박스
- 낱개
- 줄
- 팩

할 수 있는 일:

- 단위 추가
- 이름 수정
- 순서 변경
- 비활성화
- 삭제

활성 단위는 삭제할 수 없습니다.
삭제하려면 먼저 비활성화해야 합니다.

## 19. 발주처 관리

발주처 관리 화면에서는 상품 등록 시 사용하는 발주처를 관리합니다.

할 수 있는 일:

- 발주처 추가
- 이름 수정
- 비활성화
- 삭제

발주처 이름을 수정하면 해당 발주처를 사용하는 상품의 발주처 이름도 함께 변경됩니다.

## 20. 직원 관리

매장 관리자는 직원 관리 화면에서 초대 링크를 생성할 수 있습니다.

사용 방법:

1. 초대할 이메일을 입력합니다.
2. 권한을 선택합니다.
3. 초대 링크를 생성합니다.
4. 링크를 복사해 카카오톡, 문자, 이메일 등으로 전달합니다.

권한:

- 관리자
- 직원

## 21. 전체 매장

마스터 계정에서만 사용할 수 있습니다.

할 수 있는 일:

- 전체 매장 목록 확인
- 새 매장 생성
- 매장 이름 확인

## 22. 전체 사용자

마스터 계정에서만 사용할 수 있습니다.

할 수 있는 일:

- 전체 사용자 확인
- 사용자가 어느 매장에 소속되어 있는지 확인
- 이름 수정
- 매장 배정 변경
- 사용자 삭제
- 직원 초대

직원 초대 사용 방법:

1. `직원 초대` 버튼을 누릅니다.
2. 초대 이메일을 입력합니다.
3. 배정 매장을 선택합니다.
4. 권한을 선택합니다.
5. 초대 링크를 생성합니다.
6. 초대 링크를 복사해 전달합니다.

## 23. 환경설정

환경설정 화면에서는 매장 운영에 필요한 설정을 관리합니다.

사용 가능한 설정:

- 정기 휴무일
- 특정 휴무일

휴무일 설정은 홈 화면의 입고 예정 계산에 영향을 줄 수 있습니다.

## 24. 자주 발생하는 상황

### 초대 링크가 유효하지 않거나 만료되었다고 나오는 경우

가능한 원인:

- 초대 링크가 만료됨
- 이미 사용된 초대 링크임
- 관리자가 새 초대 링크를 생성하면서 기존 초대가 취소됨

해결 방법:

- 관리자에게 새 초대 링크를 요청합니다.

### 회원가입 후 매장 연결이 필요하다고 나오는 경우

가능한 원인:

- 초대 링크를 거치지 않고 일반 회원가입을 함
- 초대받은 이메일과 가입 이메일이 다름
- 초대 수락 과정이 완료되지 않음

해결 방법:

- 초대 링크를 다시 열고 초대받은 이메일로 로그인합니다.
- 문제가 계속되면 관리자에게 새 초대 링크를 요청합니다.

### 상품이 부족 재고에 계속 남아 있는 경우

가능한 원인:

- 총재고가 여전히 최소재고 이하임
- 프레시상품으로 선택되어 있음

해결 방법:

- 실제 입고 수량을 재고 작업에서 저장합니다.
- 프레시 품목이면 `입고 완료`를 누릅니다.

### 잘못 입고 완료를 누른 경우

해결 방법:

- 부족 재고 화면의 `입고완료 되돌리기` 버튼을 누릅니다.

### 잘못 재고 작업을 저장한 경우

해결 방법:

- 재고 작업 화면에서 히스토리 버튼을 누릅니다.
- 되돌릴 시점을 선택합니다.

## 25. 운영 시 권장 사용 흐름

### 아침 입고 처리

1. 바코드 스캔
2. 상품 확인
3. `입고` 선택
4. 창고 또는 매장 선택
5. 수량 입력
6. 저장

프레시 품목은 입고 처리 시 부족 재고에서 자동 제거됩니다.

바코드 입고를 누락한 경우 부족 재고 화면에서 `입고 완료`를 눌러 제거합니다.

### 영업 중 재고 확인

1. 재고현황 열기
2. 카테고리 또는 검색으로 품목 찾기
3. 창고/매장 수량 확인
4. 필요 시 재고 작업으로 이동

### 발주 확인

1. 부족재고 열기
2. 부족 품목 확인
3. 발주 후 `발주 완료` 체크
4. 긴급 품목은 `긴급발주요청` 등록

### 마감 전 확인

1. 작업로그 확인
2. 금일 입고품목 확인
3. To do list 완료 여부 확인
4. 인수인계 작성
